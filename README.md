# Zaaiwijzer iCal Feeds

Live iCal feeds voor de ZaaiKompas Zaaikalender 2025 (Benelux Binnenland).

## Abonneren

### Alle groenten (1038 events)
`
https://bobshots.github.io/Zaaiwijzer_ICS/zaaikalender_2025_alle_groenten.ics
`

### Per groente
`
https://bobshots.github.io/Zaaiwijzer_ICS/per_groente/{groente}.ics
`

## Hoe abonneren?

- **Google Calendar**: Instellingen > Agenda toevoegen > Via URL
- **iPhone/iPad**: Instellingen > Agenda > Accounts > Abonneer kalender
- **Outlook**: Agenda > Agenda toevoegen > Van internet
- **Android**: Google Calendar app > Instellingen > Agenda toevoegen > Via URL

## Wat zit erin?

Elk agenda-item bevat:
- Voorzaaien / Direct zaaien / Uitplanten / Oogsten
- De volledige keten per batch (bij zaai-events zie je direct wanneer je kunt oogsten)
- Tips voor het juiste moment
- Betrouwbaarheidsindicatie per batch

## Live feed

Deze kalender wordt automatisch bijgewerkt. Wijzigingen komen vanzelf door bij geabonneerde gebruikers.
